# Bored Again Company Website

This is the official business website for **Bored Again**, the registered business entity for Apple Developer Program enrollment.

## Setup Instructions

### Option 1: GitHub Pages (Recommended)

1. **Create a new GitHub repository** (e.g., `boredagain-website` or `boredagain-company`)

2. **Upload these files** to the repository:
   - `index.html`
   - `privacy-policy.html`
   - `README.md`

3. **Enable GitHub Pages**:
   - Go to repository Settings → Pages
   - Select source branch (usually `main`)
   - Save

4. **Your website will be available at**:
   - `https://yourusername.github.io/boredagain-website/`

### Option 2: Custom Domain

If you want to use a custom domain like `boredagain.com` or `boredagain.com.au`:

1. **Purchase the domain** from a registrar (Namecheap, GoDaddy, etc.)

2. **Deploy to GitHub Pages** (as above)

3. **Configure DNS**:
   - Add a CNAME record pointing `www` to `yourusername.github.io`
   - Or add A records for the root domain (see GitHub Pages documentation)

4. **Add custom domain in GitHub Pages settings**

### Option 3: Other Hosting Services

You can also host these files on:
- **Netlify**: Drag and drop the folder
- **Vercel**: Connect GitHub repository
- **Any static hosting service**
